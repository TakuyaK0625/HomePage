【KAKEN】データのクリーニング（４）

ー各年度配分額列の整理ー

Takuya Kubo, 2020/05/20


０．はじめに

このページでは、主にdplyrパッケージとstirngrパッケージ、tidyrパッケージを用いて各年度配分額列を整理する方法を紹介していきます。具体的には、以下をゴールに進めていきます。

  1. 各年度の配分額をそれぞれ１行で表示する
  2. 各年度の配分額をそれぞれ１列で表示する


１．環境設定

必要なパッケージとデータは以下の通りです。

# インストールされていない場合は、「#」から始まるコマンドの「#」を外して先に実行してください。
# install.packages("data.table")
# install.packages("dplyr")
# install.packages("DT")
# install.packages("stringr")
# install.packages("tidyr")

# パッケージの読み込み
library(data.table)   # 高速なデータインポートのため
library(dplyr)        # データの整理のため
library(DT)           # 集計表の出力のため
library(stringr)      # 文字列操作のため
library(tidyr)

# 科研費データの読み込み
d <- fread("kaken_2020.csv")


２．データの確認

まず、必要な列を選択し、各年度配分額の列にどのようなパターンで値が入っているのかを確認します。以下の表からは、各年度の配分額が「年度:配分額」で表されていることが分かります。また、研究期間が複数年度にまたがる場合にはスペースが各年度の配分額を区切っていることが見て取れるのではないでしょうか。

# 必要な列だけ選択し、名前も変えておく
D <- d %>% select(研究課題番号 = `研究課題/領域番号`, 各年度配分額)

# 最初の100行を表示
datatable(D[1:100,])


２．各年度の配分額をそれぞれ１行で表示する

それでは、各年度の配分額がそれぞれ１行で表示されるように各年度配分額を処理していきます。基本的な発送としては研究分担者列を整理したこちらのページと同じです。まず、stringrパッケージのstr_split関数を用いて各年度配分額の値をスペースを区切りに分割します。各年度配分額列の最初の5行を見てみると、年度で分割された値が埋め込み形式でそれぞれの行の中に入っていることが分かるかと思います。

D1 <- D %>% 
    
    # 各年度配分額の値をスペースを区切りに分割
    mutate(各年度配分額 = str_split(各年度配分額, " ")) 

# 各年度配分額列の最初の５つの値
D1$各年度配分額[1:5]
## [[1]]
## [1] "2020:2730000" "2021:390000"  "2022:390000"  "2023:780000" 
## 
## [[2]]
## [1] "2020:3120000" "2021:130000"  "2022:520000"  "2023:390000" 
## 
## [[3]]
## [1] "2020:1690000" "2021:780000"  "2022:1170000" "2023:520000" 
## 
## [[4]]
## [1] "2020:780000" "2021:130000"
## 
## [[5]]
## [1] "2020:1690000" "2021:910000"  "2022:910000"


次に、この埋め込み形式を解除して配分額が各年度で1行になるようにします。この時に使うのがtidyrパッケージのunnest関数であり、埋め込み形式を解除したい列名を入れたらOKです。なお、埋め込みが解除された場合には、残りの列は全てオリジナルのデータがコピーされます。

# 各年度配分額列の埋め込み形式を解除
D2 <- D1 %>% unnest(各年度配分額)

# 最初の５行だけ表示
datatable(D2[1:500,])


各年度配分額列には年度情報と配分額情報の2つの情報が混在しているため、これらを別々の列に分けていきます。この時に便利なのがtidyrパッケージのseparate関数です。separate関数は分割したい列名（col）と分割後の列名（into）、分割する際の目印（sep）を指定してあげればOKです。

# 各年度配分額列の値を「：」で分割し、「年度」列と「配分額」列に入れる
D3 <- D2 %>% separate(col = 各年度配分額, into = c("年度", "配分額"), sep = ":")

# 最初の500行だけ表示
datatable(D3[1:500,])


こレデ十分に綺麗なデータができましたが、折角なので、あと１つだけ情報を加えてみます。複数年度のデータを合わせた場合に、年度ではなく何年目かで集計することがあるかもしれません。何年目かのデータを得るためには、研究課題ごとにグループ化を行い、それぞれのグループにおいて1から番号を振っていきます。

D4 <- D3 %>% 
    group_by(研究課題番号) %>%  # 研究課題番号でグループ化
    mutate(年目 = 1:n())        # 各グループで「年目」列に１から番号を振る
    
# 最初の500行を表示
datatable(D4[1:500,])


３．各年度の配分額をそれぞれ１列で表示する

各年度の配分額をそれぞれ１列で表示させるには、tidyrパッケージのpivot_wider関数を用いて、縦持ちのデータ形式を横持ちのデータ形式に変換してあげれば良いです。縦持ち変換、横持ち変換の詳細については、年度別の配分額を集計したこちらのページをご参照ください。以下のコードでは、年度列の各値（2018, 2019, 2020）を新しい列名に指定し、その値として配分額列を指定しています。

D5 <- D4 %>%
    
    # 研究課題番号はそのまま列として残し、年度を列名、配分額を値に横持ち変換
    pivot_wider(id_cols = 研究課題番号, names_from = 年度, values_from = 配分額)

# 最初の100行を表示
datatable(D5[1:100,])


４．おわりに

今回は各年度配分額列を用いて、配分額を年度ごとに行、列で分ける方法を紹介いたしました。これができると、年度ごとの集計ができたり、研究者の移動情報にも対応できたりと、より詳細な集計ができるようになるかと思います。ここから集計を行う際の注意点としては、これまで配分額を文字列として扱ってきたため足したり引いたりができません。集計の際にはas.numeric関数などを用いて、あらかじめデータ型を変換するようにしてください。
