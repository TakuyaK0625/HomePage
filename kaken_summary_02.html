【KAKEN】データの要約（２）

ー配分額と採択件数を相対的に表現ー

Takuya Kubo, 2020/05/21


０. はじめに

前回は研究機関や審査区分を軸に配分額と採択件数を集計しましたが、今回はこれらの値を以下の３通りの方法で相対的に表現するコードを紹介していきます。

  • 研究機関ごとの配分額と採択件数の順位
  • 研究機関ごとの配分額と採択件数の百分率
  • 研究機関ごとの配分額と採択件数を常用対数＋正規化


１. 環境設定

必要なパッケージとデータは以下の通りです。今回は2020年度の全採択課題を用います。

# インストールされていない場合は、以下のコマンドの「#」を外して実行
# install.packages("data.table")
# install.packages("dplyr")
# install.packages("DT")
# install.packages("stringr")

# パッケージの読み込み
library(data.table)   # 高速なデータインポートのため
library(dplyr)        # データの整理のため
library(DT)           # 集計表の出力のため
library(stringr)      # 文字列操作のため

# 使用するデータのインポート
d <- data.table::fread("kaken_2020.csv")


２. データの確認と集計

前回と同様に「研究機関」、「研究種目」、「審査区分」「総配分額」の４列のみに絞って確認します。また、データ量が多いので、本ページではランダムに抽出した1000行のみを表示しておきます。

D <- d %>%
  select(研究機関, 研究種目, 審査区分, 総配分額) # 列の絞り込み

D %>% 
  sample_n(1000) %>%   # ランダムに1000行抽出
  datatable()                 # 可視化


データの集計方法についても前回と同様です。研究機関ごとの配分額と採択件数をdplyrパッケージのgroup_by関数とsummarize関数を用いて算出します。

D1 <- D %>% 

  # 総配分額をnumeric型に変換
  mutate(総配分額 = as.numeric(総配分額)) %>%
  
  # 研究機関ごとにグループ化
  group_by(研究機関) %>%
  
  # 各グループ（機関）内で総配分額の合計と行数を計算
  summarize(配分額 = sum(総配分額), 採択件数 = n())
  
# 可視化
datatable(D1)


３. 順位

順位を取得するためにはrank関数を用います。rank関数は小さい値から１位、２位と順位を返すので、配分額にはマイナスをつけて大小関係を逆転させています。また、ties.methodは同じ値（タイ）だった場合の対処法を指定しており、ここでは最も値の小さい順位となるようにしています。例えば、８位の機関の下に同値の機関が３機関ある場合、このグループを９位として扱うような感じです。

D1 %>%
  mutate(配分額ランク = rank(-配分額, ties.method = "min")) %>%
  
  mutate(採択件数ランク = rank(-採択件数, ties.method = "min")) %>%
  
  datatable()


４. 百分率

百分率を算出するにはdplyrパッケージのpercent_rank関数を用います。percent_rank関数も小さい値から0%⇨100%と百分率を返すのですが、今回は配分額が最も多い機関を100%としたいのでマイナスはつけません。

D1 %>%
  mutate(配分額百分率 = round(100 * percent_rank(配分額), 2)) %>%
  
  mutate(採択件百分率 = round(100 * percent_rank(採択件数), 2)) %>%
  
  datatable()


５. 常用対数＋正規化

最後に、以下の文献で提案されている可視化手法の前提となる集計方法を紹介いたします。具体的な集計方法としては、配分額や採択件数を常用対数に変換し、最大値となる機関が４になるように正規化するというものです。この変換により、１の差は実際の値の10倍、1/10に相当することになります。


調麻佐志（2018）「科学研究費補助事業採択データによる大学の強みや多様性の可視化」『特別研究促進費「研究力分析指標プロジェクト」報告書（2016-2017年度）』


以下のコードでは、log10関数を用いて配分額を常用対数に変換し、正規化した後で小数点第３位を四捨五入にしています。なお、この変換方法ではマイナスの値を取ることがありますが、それを避けるには最大値を５にしてあげるなどの方法があります。以下のコードの４を５に変えるだけです。

D1 %>%
  mutate(配分額対数 = round(log10(配分額)-max(log10(配分額)-4), 2)) %>%
  
  mutate(採択件数対数 = round(log10(採択件数)-max(log10(採択件数)-4), 2)) %>%
  
  datatable()


６. おわりに

今回は機関ごとの配分額を相対的に表現するための３つの方法を紹介していきました。最近は何かとランキングが蔓延っていますので、この中でも比較的によく使っているような気がします。百分率は雑誌や論文の被引用数でお馴染み思いますが、科研費で使う場合には「国立大学」や「私立大学」というグループで対象を絞った方がわかりやすいかもしれません。最後に紹介した方法は、本来、分野別に算出して各大学のレーダーチャートを書くためのものです。肝心のレーダーチャートを描画する方法については可視化のページで改めて紹介したいと思います。
